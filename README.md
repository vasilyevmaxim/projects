# projects
Список документации и решений, сделанных мной в процессе работы преподавателем.

## Состав репозитория
- AccreditationTask - сформированное практико-ориентированное задание для аккредитации колледжа;
- EduTest - система онлайн тестирования, написанная с использованием Blazor API, поддерживает написание разделов, тестов, лекций. Реализованы анализ оценок по группам и по студентам, возможность администрирования системы под ролью преподавателя;
- EfficiencyCalculator - десктопное приложение, написанное на WPF, предназначенный для автоматизации формирования отчетности по эффективности сообщества;
- MethodicalDocumentation - список некоторых сформированных мной документов, включающий план проведения урока, комплектны оценочных материалов;
- SQLTasks - список практических работ, включающий теоретические основы и практическую часть, написанные полностью мной, для освоения студентами учебной программы. Включает работы по MSSQL и PostgreSQL;
- UML - некоторые презентации и практическая работа для освоения основ UML моделирования студентами.
